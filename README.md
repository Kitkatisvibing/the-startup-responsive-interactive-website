# Titel
ASCII format converter
Ik heb een design voor een pagina om ASCII naar het format dat soundslice gebruikt te converten.

## Beschrijving
Ik heb besloten een simpeler design te gebruiken, om beter aan te sluiten met de huisstijl.

Ik heb 3 layouts gemaakt. Klein, middel, en groot scherm. Het verschil tussen die eerste twee is subtiel. Het grote scherm heeft een duidelijke nieuwe layout.

Ik heb ervoor gezorgd dat alle elementen op de website met toetsenbord genavigeerd kunnen worden. Ik dacht eraan om de kleur oranje te veranderen voor leesbaarheid, maar vanwege de huisstijl besloot ik dat niet te doen. Bij een project waar ik meer tijd voor had zou ik dit hebben opgelost met een hoge contrast optie.

Als interactie besloot ik om de convert button bruikbaar te maken. Als je op de knop drukt, dan verandert de tekst in de textarea


## Kenmerken
Ik heb gebruik gemaakt van het js 3 stappenplan, en heb zelf online code gevonden om de tekst in de textareas te wijzigen.
